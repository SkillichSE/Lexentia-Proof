About ModelArena

How we test and rank free AI models every day

What is ModelArena?

ModelArena is an open-source platform that automatically benchmarks free AI models every 24 hours. We test real performance—speed, code quality, reasoning, and more—so you can make informed decisions without hype or marketing fluff.

All tests run via GitHub Actions. All code is public. All data is reproducible. Zero cost, zero bias.

How It Works

⏰ GitHub Actions triggers at 3:00 UTC
→
🤖 Python script tests all models
→
💾 Results saved to JSON
→
📊 Site updates automatically

Every day at 3:00 AM UTC, our GitHub Actions workflow spins up, runs identical prompts across all free models, measures speed and quality, then commits the results. The site pulls fresh data on every visit.

No manual intervention. No bias. Just automated, consistent testing.

What We Test

⚡ Speed
Tokens per second (tok/s) across short, medium, and long prompts. Faster = better for interactive use.
💻 Code Quality
Can it write working code? Syntax correctness, comments, logic. Tested with Python functions of varying complexity.
🧠 Reasoning
Logic puzzles, math problems, common-sense questions. Does it actually think or just pattern-match?
📋 Instruction Following
Can it follow multi-step instructions accurately? Critical for real-world tasks.
🌐 Translation
English ↔ Russian, English ↔ Spanish. Accuracy and fluency compared against reference translations.

Models We Test

We focus exclusively on free-tier models with generous limits. Current lineup includes:

  • Groq — Llama 3.1 8B, Llama 3.3 70B, and more (blazing fast inference)
  • Google — Gemini 1.5 Flash, Gemini 1.5 Pro (huge context windows)
  • More coming — we add models as they become available for free

Models are added when: (1) they're free without credit card, (2) API is publicly accessible, (3) rate limits allow daily testing.

Limitations & Caveats

No benchmark is perfect. Here's what ours doesn't cover:

  • Long-form quality — our prompts are short by design (speed testing)
  • Multimodal — text-only for now (vision/audio coming later)
  • Domain-specific — we test general ability, not medical/legal/scientific expertise
  • Subjective quality — style, creativity, tone are hard to quantify

Use our data as one signal, not the only signal. Always test models yourself for your specific use case.

FAQ

How often do results update?
Every 24 hours at 3:00 AM UTC via GitHub Actions.
Can I see the test prompts?
Yes! All prompts are in src/config.py in our GitHub repo.
Why are some models missing?
We only test models with free access (no credit card required) and public APIs.
Can I run benchmarks myself?
Absolutely. Fork the repo, add your API keys to GitHub Secrets, enable Actions. Done.
How do you calculate overall scores?
Weighted average of speed, code, reasoning, instruction, translation. Exact weights in src/config.py.
Who runs this?
Open-source community project. Anyone can contribute via GitHub.
