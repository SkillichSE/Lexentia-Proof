What is ModelLens?

An open-source benchmark that tests free AI models every 24 hours. We measure real performance—speed, code quality, reasoning—so you can choose the best model for your needs without marketing hype.

What We Test

💻

HumanEval · Code

Real code execution. Functions are run against test cases—syntax and logic must pass.

🧮

GSM8K · Reasoning

Grade-school math word problems and logic puzzles. Verified correct answers only.

📚

MMLU · Knowledge

Instruction following, format compliance, and multi-domain knowledge tests.

🌐

Translation

English ↔ Russian, English ↔ Spanish accuracy via script and vocabulary detection.

⚡

Throughput

Tokens per second averaged over short, medium, and long prompts.

🔄

Updated Daily

Automated via GitHub Actions. Fresh data every 24 hours.

Why Size Categories Matter

Comparing an 8B model to a 405B model isn't fair. They're built for different use cases. Our 3 categories ensure you find the best model for your constraints:

  • Small (≤10B) — 100–300+ tok/s · low memory · edge/mobile friendly
  • Medium (10–50B) — 30–100 tok/s · balanced performance and resource use
  • Large (50B+) — 5–40 tok/s · maximum capability · needs serious hardware
