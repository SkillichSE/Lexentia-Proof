What is ModelLens?

An open-source benchmark that tests free AI models every 24 hours. We measure real performance—speed, code quality, reasoning—so you can choose the best model for your needs without marketing hype.

What We Test

💻

Code Quality

Real code execution. Does it actually run? Syntax, logic, edge cases.

🧠

⚡

English ↔ Russian, English ↔ Spanish accuracy.

Why Size Categories Matter

Comparing an 8B model to a 120B model isn't fair—it's like comparing a motorcycle to a truck. They're built for different use cases. Our categories ensure you find the best model for your constraints:

  • Small (≤10B) — Fast, low memory, edge/mobile friendly
  • Medium (10–50B) — Balanced performance and resource use
  • Large (50B+) — Maximum capability, needs serious hardware
